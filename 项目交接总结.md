## 7. 启动方式

### 7.1 Redis

用户本机 Redis 目录曾经使用：

```text
D:\develop\Redis-x64-3.2.100
```

启动命令示例：

```powershell
cd D:\develop\Redis-x64-3.2.100
.\redis-server.exe .\redis.windows.conf
```

Redis 密码请写在本地配置或环境变量中，不要写入公开文档。

### 7.2 MySQL

数据库名：

```text
farm_system
```

初始化脚本：

```text
mysql/farm_system.sql
```

MySQL 密码请写在本地配置或环境变量中，不要提交到公开仓库。

### 7.3 后端

推荐使用项目里的脚本：

```powershell
cd springboot
.\start-backend.ps1
```

这个脚本的作用：

- 编译后端。
- 生成运行 classpath。
- 使用 `java -cp ... com.example.SpringbootApplication` 启动。

后端默认地址：

```text
http://localhost:9090
```

### 7.4 前端

```powershell
cd vue
npm install
npm run dev
```

默认地址：

```text
http://127.0.0.1:5173/
```

构建命令：

```powershell
npm run build
```

## 8. 本地配置文件方案

公开仓库提交：

```text
springboot/src/main/resources/application-local.example.yml
```

本地真实配置：

```text
springboot/src/main/resources/application-local.yml
```

真实配置已被 `.gitignore` 忽略，不应提交。

可以把 example 复制一份为 local，然后写入本机配置：

```yaml
spring:
  datasource:
    username: root
    password: your_mysql_password
  data:
    redis:
      password: your_redis_password

file:
  upload-dir: D:/farm-system/uploads
```
